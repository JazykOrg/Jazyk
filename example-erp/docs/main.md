# Warehouse ERP

This software is a warehouse management system that allows you to manage the ACME warehouse.

## Frontend

The frontend to the Warehouse system is a web application built using React and TypeScript.

See the [frontend documentation](./frontend.md) for more information.

## Backend

The backend to the Warehouse system is a REST API built using Node.js and Express.

See the [backend documentation](./backend.md) for more information.
