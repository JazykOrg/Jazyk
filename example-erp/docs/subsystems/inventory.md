# Inventory Management

The inventory system manages a set of products and their stock levels.

# Properties

Each product inventory item has the following properties:
- `id` - a unique identifier for the product
- `name` - the name of the product
- `stock` - the current stock level of the product

# Operations

The inventory system supports the following operations:

- `addProduct` - adds a new product to the inventory
- `updateStock` - updates the stock level of a product

# Security

All operations can be performed by warehouse staff only.
